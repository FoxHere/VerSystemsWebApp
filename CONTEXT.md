# VerSystems - Contexto do Projeto

## Arquitetura
- MVVM com GetX
- GoRouter para navegação
- Firebase (Auth, Firestore, Storage)
- Fx UI (biblioteca customizada)

## Estrutura
- web_2025: Aplicação principal
- core_2025: Package com funcionalidades base
- fx_ui: Biblioteca de componentes

## Padrões
- ViewModels reativos com GetxController
- Repositories encapsulam lógica de negócio
- Services fazem integração com APIs
- Either para tratamento de erros
